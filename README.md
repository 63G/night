# Threadlight Date Night

A production-ready guided long-distance date-night web app. It includes a flower bloom opener, would-you-rather prompts, a memory card game, a custom Wordle round, a Van Gogh-inspired generated painting moment, battle-ships, and a letter-writing keepsake that can be downloaded as a PNG.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The generated painting asset is stored at `public/art/starry-thread.png`.
